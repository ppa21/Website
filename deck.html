About the App

Tokimon Deck is a Client-Server application built using Java, JavaFx and Spring Boot which utilizes REST API and HTTP requests to create, store and retrieve rare fictional creatures called Tokimon. These Tokimons are created by users in the application giving them their unique attribues such as height, color, power and more which would allow them to survive the wilderness of Korea.
